Euler's Constant and its Applications

Subject: Calculus
Year: 2019

Summary

A simple paper that I wrote about Euler's constants and its applications as a part of my high-school graduation thesis for which I got the highest grade. The paper talks about Leonard Euler's discovery of the natural constant (which was later named after him as Euler's constant e). The paper also mentions Jacob Bernoulli and his contribution to the discovery of the constant. Finally, the paper discusses some of the applications of the constant and some functions related to it to calculate equations in many scientific fields such as biology, geography and physics.

Parser and Evaluator of Complex Polynomials

Subject: Data Structures
Programming Language: C++
Year: 2021

Summary

This project was a part of my 2nd year course Data Structures. Using C++ I implemented a custom Polynomial structure which also works with complex coeficients accompanied with all of the essential constructors and methods. Basic operators were also overloaded so that mathematical operations (such as addition, subtraction, multiplication and exponentiation) can be used. Besides of the mentioned methods I also implemented a parser and an evaluator so that a user can input any type of polynomial and a mathematical operation in the form of a string and be given the result of the input also in the form of a string. If the input is faulty then the result is a text which indicates where a mistake could have happened in the input.

Randomized Self-Balancing Binary Search Tree Structure

Subject: Data Structures
Programming Language: C++
Year: 2022

Summary

This project was a part of my 2nd year course Data Structures. Using C++ I implemented a custom binary search tree structure accompanied with all of the essential constructors and methods. The BST structure was also implemented in a way so that it is self-balancing, meaning that the binary search tree automatically keeps its height (maximal number of levels below the root) small in the face of arbitrary item insertions and deletions. This was acomplished by making the structure as a combination of a standard BST and a heap. Each new node of the tree was also given a random number and the tree was then balanced by simple rotations so that the node which has the biggest random number attribute is the root of the tree. It can be shown that for a big enough tree the height of the tree remains relatively small due to the randomness of the additional numbers in each node and the 'left-rigth', 'right-left' rotations used to balance the tree out.
